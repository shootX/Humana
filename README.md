# Humana - рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ

## рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

- ­ЪЌ║№ИЈ Google Maps рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃўрЃА рЃџрЃЮрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ
- ­ЪЊ▒ рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў
- ­ЪЉц рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- ­ЪЊі рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ

## рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

## рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ

1. рЃЊрЃљрЃљрЃЎрЃџрЃЮрЃюрЃўрЃарЃћрЃЌ рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃљ:
```bash
git clone https://github.com/shootX/Humana.git
```

2. Android Studio-рЃерЃў рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў

3. Google Maps API Key-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ:
   - рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ [Google Cloud Console](https://console.cloud.google.com/)
   - рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃю рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў
   - рЃЕрЃљрЃарЃЌрЃћрЃЌ Maps SDK for Android
   - рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ Credentials рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ
   - рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў API Key (Create Credentials -> API Key)
   - рЃерЃћрЃќрЃдрЃБрЃЊрЃћрЃЌ API Key рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ Android рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА:
     - Application restrictions -> Android apps
     - рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА SHA-1 рЃЊрЃљ package name

4. API Key-рЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ:
   - рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ `app/src/main/AndroidManifest.xml`
   - рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃћрЃЌ `com.google.android.geo.API_KEY` meta-data рЃЌрЃћрЃњрЃў
   - рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў API Key value рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў:
   ```xml
   <meta-data
       android:name="com.google.android.geo.API_KEY"
       android:value="рЃЌрЃЦрЃЋрЃћрЃюрЃў_API_KEY" />
   ```

5. SHA-1 рЃ«рЃћрЃџрЃЏрЃЮрЃгрЃћрЃарЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ:
   ```bash
   cd android
   ./gradlew signingReport
   ```

6. рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ Android Studio-рЃЊрЃљрЃю

1. Google Maps рЃљрЃа рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ:
   - рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ API Key-рЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃћ
   - рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ Maps SDK for Android рЃЕрЃљрЃарЃЌрЃБрЃџрЃўрЃљ

   - рЃњрЃљрЃЊрЃљрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ SHA-1 рЃ«рЃћрЃџрЃЏрЃЮрЃгрЃћрЃарЃљ Google Cloud Console-рЃерЃў
